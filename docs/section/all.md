# L_lib

`L_lib.sh` is a comprehensive bash library designed to simplify shell scripting by providing a rich set of helper functions and utilities. It aims to bring features from other programming languages to bash, making it more powerful and easier to use for complex tasks.

## Structure

The library is organized into the following sections:

*   [**all**](all.md): This section. Provides an overview of the entire library.
*   [**ansi**](ansi.md): Functions for controlling ANSI escape codes for cursor movement and text formatting.
*   [**argparse**](argparse.md): A powerful argument parser, similar to Python's argparse.
*   [**args**](args.md): Utilities for working with script arguments.
*   [**array**](array.md): A collection of functions for array manipulation.
*   [**asa**](asa.md): Functions for ....

*   [**finally**](finally.md): Functions for...
*   [**func**](func.md): Meta-functions for manipulating other functions.
*   [**globals**](globals.md): Global variables used by the library.

*   [**json**](json.md): Functions for working with JSON data.
*   [**lib**](lib.md): Functions for...

## Usage

To use the library, simply source it in your script:

```bash
source /path/to/L_lib.sh
```

Then you can use any of the functions provided by the library.

You can also see the [bin/L_lib.sh](bin/L_lib.sh) file to check the code.
